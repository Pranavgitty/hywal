# Wallpaper Switcher

Fast, daemon-powered wallpaper switcher for Hyprland built with Quickshell and Rust.

## Features

- Instant popup
- Persistent Rust daemon
- Keyboard driven
- Mouse support
- Matugen compatible
- Caelestia wallpaper integration
- Configurable wallpaper directory
